# ADmyBRAND Dashboard

A simple *Next.js 15 + Tailwind CSS* based dashboard built for analytics and insights.

## 🚀 Features
- 📊 Metric Cards
- 📈 Line Chart
- 📊 Bar Chart
- 🥧 Pie Chart
- 📋 Data Table
- 🎨 Responsive UI with Tailwind CSS

## 🛠 Tech Stack
- Next.js 15 (App Router)
- React
- Tailwind CSS
- Recharts

## ▶ How to Run Locally
```bash
npm install
npm run dev
